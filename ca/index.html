Conceptes Bàsics de HTML:5

HTML:

Estructura bàsica del HTML

!DOCTYPE html: Es el tipus de document

html: Tota la informació que hi hem de posar a la web comença amb aquesta etiqueta

head: Es l'informació de la web NO VISIBLE

/head: S'ultilitza per tancar el head

body: Dins va tot el contingut de la Web VISIBLE

/body: S'ultilitza per tancar el body

/html: S'ultilitza per tancar el html

CSS:

Atributs CSS

Els atributs proporcionan informació adicional sobre els elements.

Un exemple d'un atribut podría ser el de la imatge → href

href: S'ultilitza per fer referència a una part del nostre códi o a algú enllaç d'internet

Etiquetes CSS

Les etiquetes o selectors serveixen per donar estil només a aquests.

Alguns exemples son: h1, #section-1, #header.

Propietats CSS

La propietats son per definir les mateixes de les reglas, per exemple la font del text, el tamany, etc.

Alguns exemplos son: color:, background-color:, font-size:.

Regles CSS

Una regla CSS es composa d'un selector y d'un bloc de declaració.

Cada declaració està composta per un o mes parells de propietat/valor

El bloc de declaració comença i acaba con los signes de claus { ... }

Cada parell propietat/valor es separen per el signe punt i coma ;

Cada propietat deu separarse del seu valor per el signe dos punts :

Comentaris CSS

Per afegir comentaris a CSS s'utilitza la següent estructura:

/* TEXT A COMENTAR */

A mes es mostrarà en color verd per confirmar que es un comentari.

(El color depén de la app que utilitzem per escriure el codi)

Valors CSS

Els Valors aporten informació adicional sobre la propietat i a mes donen format
