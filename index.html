Conceptos Básicos de HTML:5

HTML:

Estructura básica del HTML

!DOCTYPE html: Es el tipo de documento

html: Toda la información que hay que poner en la web se empieza mediante esta etiqueta

head: Es información de la web NO VISIBLE

/head: Se usa para cerrar el head

body: Ahí dentro va todo el contenido de la Web VISIBLE

/body: Se usa para cerrar el body

/html: Se usa para cerrar el html

CSS:

Atributos CSS

Los atributos proporcionan información adicional acerca de los elementos.

Un ejemplo de un atributo podría ser el de la imagen → href

href: Se usa para hacer referencia a una parte de nuestro código o a algún enlace de internet

Etiquetas CSS

Las etiquetas o selectores sirven para dar estilo solo a esos.

Algunos ejemplos son: h1, #section-1, #header.

Propiedades CSS

La propiedades son para definir las mismas de las reglas, por ejemplo la fuente del texto, el tamaño, etc.

Algunos ejemplos son: color:, background-color:, font-size:.

Reglas CSS

Una regla CSS se compone de un selector y de un bloque de declaración.

Cada declaración está compuesta por uno o más pares propiedad/valor

El bloque de declaración empieza y acaba con los signos de llaves { ... }

Cada par propiedad/valor se separa por el signo punto y coma ;

Cada propiedad debe separarse de su valor por el signo dos puntos :

Comentarios CSS

Para añadir comentarios en CSS se usa la siguiente estructura:

/* TEXTO A COMENTAR */

Además se mostrará en color verde para confirmar que es un comentario.

(El color depende de la app que usemos para escribir el código)

Valores CSS

Los Valores aportan información adicional sobre la propiedad y además dan formato
